Data Analytics

A correlation analysis of potential relationships between selected stock closing prices and daily subreddit activity.

Learn More

Through python code for collection and cloud services for automation, closing prices for selected stocks as well as activity data for selected subreddits are gathered daily and submitted to an excel file. Calculations are then done to polish data, determine changes for each data point and format findings accordingly. Then utilizing JavaScript and native features, a chart is dynamically generating for each possible combination of data based off of a correlation matrix allowing for easy visualization of findings.
